## Running The Cracker

### Syntax

`fohmd5.py path/to/hash.txt -a dict -d path/to/dictionary.txt`

`fohmd5.py path/to/hash.txt -a brute`

## Options
### Attack types

- dict (Dictionary)
- brute (Brute-Force)
